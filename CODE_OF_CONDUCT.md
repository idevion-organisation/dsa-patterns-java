# Code of Conduct

## Our Community

DSA Patterns Java by Idevion is a beginner-friendly open-source learning project.

This repository is created to help students learn Data Structures and Algorithms in Java using a pattern-based approach.

We want this project to remain helpful, respectful, and safe for everyone.

Everyone participating in this project is expected to follow this Code of Conduct.

---

## Our Standards

Examples of positive behavior include:

- Being respectful to other contributors
- Helping beginners with patience
- Giving clear and constructive feedback
- Explaining suggestions politely
- Respecting different learning speeds
- Keeping discussions focused on improving the project
- Using inclusive and professional language

---

## Unacceptable Behavior

The following behavior is not allowed:

- Harassment, bullying, or personal attacks
- Insulting beginners for asking simple questions
- Disrespectful or offensive language
- Spam or irrelevant self-promotion
- Sharing copied content without credit
- Adding misleading, incorrect, or low-quality content intentionally
- Creating unnecessary arguments in issues or pull requests
- Discrimination based on identity, background, experience level, or personal beliefs

---

## Contribution Discussions

When discussing issues, pull requests, or improvements:

- Focus on the content, not the person
- Explain what should be improved and why
- Be specific and helpful
- Avoid rude or aggressive comments
- Respect maintainers' final decisions
- Keep the learning goal of the repository in mind
