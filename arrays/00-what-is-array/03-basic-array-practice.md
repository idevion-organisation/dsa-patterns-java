# 03. Basic Array Practice

Practice these after learning:

- 1D array
- 2D array
- indexing
- traversal
- basic complexity

Recommended order:

```txt
Manual Problems → LeetCode → CodeChef / Codeforces
```

---

## Practice Goals

You should be able to:

- print array elements
- find sum
- find max/min
- count elements
- search an element
- traverse 2D array
- avoid index mistakes

---

## Manual Practice Problems

### 1. Print Array

```txt
Input: [10, 20, 30]
Output: 10 20 30
```

Pattern: Traversal

---

### 2. Sum of Array

```txt
Input: [1, 2, 3, 4]
Output: 10
```

Pattern: Traversal

---

### 3. Maximum Element

```txt
Input: [5, 9, 2, 11]
Output: 11
```

Important:

```java
int max = arr[0];
```

Do not start with `0`, because array may contain negative numbers.

---

### 4. Count Even Numbers

```txt
Input: [2, 5, 8, 9, 10]
Output: 3
```

Condition:

```java
arr[i] % 2 == 0
```

---

### 5. Search an Element

```txt
Input: arr = [4, 7, 1, 9], target = 7
Output: true
```

Pattern: Linear Search

---

### 6. Print 2D Matrix

```txt
Input:
1 2
3 4

Output:
1 2
3 4
```

Pattern: Nested loops
