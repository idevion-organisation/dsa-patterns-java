# 03. Day 2 Practice

This practice is not about writing complex code.

The goal is to learn how to identify the pattern before solving.

---

## Practice Method

For every problem, write:

```txt
Given:
Asked:
Pattern:
Why this pattern:
Time Complexity:
Space Complexity:
```

Example:

```txt
Problem: Find maximum element in array

Given: array of integers
Asked: maximum value
Pattern: Traversal
Why: need to check every element
Time: O(n)
Space: O(1)
```
