## How to Study

1. Read the concept file.
2. Understand the pattern signal.
3. Dry run manually.
4. Study Java templates.
5. Solve practice problems.
6. Revise mistakes and checklist.

## Final Goal

The goal is not to memorize array solutions.

The goal is to identify patterns and solve new problems confidently.
